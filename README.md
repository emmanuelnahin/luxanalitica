# luxanalitica es la intersección entre la precisión matemática y el valor sustancial del negocio, utilizando la herramienta más universal del mundo: Excel.

En luxanalitica transformamos el caos de las filas y columnas en claridad estratégica. Creemos que no hace falta software complejo para entender un negocio; hace falta maestría en la herramienta que todos ya tienen: Excel.
